📖 使用说明

        

        

          
第一步：在顶部输入框粘贴你的 DeepSeek API key，点击「配置」。

          
第二步：切换到「回测实验室」tab，调整参数并运行一次回测。

          
第三步：回到「AI 分析」tab，点击「分析最近回测」或「流式分析」。

          
第四步：在下方对话框中自由提问，AI 会基于回测结果给出专业建议。

          

            💡 提示：API key 仅在当前服务器进程内存中保留，关闭 UI 后不会保存。推荐通过环境变量 DEEPSEEK_API_KEY 持久化配置。
